# E-commerce-project
This project focuses on analyzing e-commerce sales data to uncover business insights,customer behaviour patterns and performance trends.
The goal is to support decision making through visualizations and data driven insights.
# Dataset Columns
-Order_ID  
-Customer_ID  
-Date  
-Age  
-Gender  
-City   
-Product_Category  
-Unit_Price  
-Quantity  
-Discount_Amount  
-Total_Amount  
-Payment_Method  
-Device_Type
-Session_Duration_Minutes
-Pages_Viewed   
-Is_Returning_Customer  
-Delivery_Time_Days  
-Customer_Rating
